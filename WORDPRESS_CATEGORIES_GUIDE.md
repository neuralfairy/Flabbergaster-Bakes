## ğŸ’¡ Tips

1. **One Category Per Post**
   - Each cupcake should have ONE category
   - If multiple are checked, it will use the first one

2. **Category Names Must Match**
   - Use exact names: "Signature Collection" not "signature collection"
   - Case matters!

3. **Default Category**
   - If no category is assigned, it defaults to "Signature Collection"

4. **Uncategorized**
   - WordPress has a default "Uncategorized" category
   - You can delete or ignore it
